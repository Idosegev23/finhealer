# 🏥 תוכנית ההבראה הפיננסית של φ (Phi)

**המפרט המלא - 6 שלבים**

---

## 📋 **מיפוי: תוכנית הבראה → Phases במערכת**

| שלב הבראה | Phase במערכת | זמן משוער | מטרה |
|-----------|--------------|-----------|------|
| 1️⃣ שיקוף | `reflection` + `data_collection` | 1-2 שבועות | תמונת מצב מלאה |
| 2️⃣ שינוי הרגלים | `behavior` | 2-4 שבועות | זיהוי דפוסים + שינוי הרגלים |
| 3️⃣ תכנון תקציב | `budget` | 1 שבוע | בניית תקציב חכם |
| 4️⃣ יעדים ומטרות | `goals` | משך זמן | הגדרת חזון ויעדים |
| 5️⃣ איחוד הלוואות | (פעולה בתוך `goals`) | לפי צורך | אופטימיזציה של חובות |
| 6️⃣ בקרה | `monitoring` | לצמיתות | שמירה על המסלול |

## 1️⃣ **שלב 1: שיקוף (Reflection)**

### **🎯 מטרה:**
> בחינה של העבר, ההווה והעתיד  
> זיהוי התנהלות קיימת  
> יצירת תמונה מלאה שממנה מתחילים

---

### **📊 תוצאה של שלב 1:**

**בסיום, יש למערכת:**
1. ✅ **Baseline** - מה המשתמש חושב שהוא מוציא (reflection)
2. ✅ **Reality** - מה הוא באמת מוציא (documents)
3. ✅ **Gap** - ההבדל בין חשיבה למציאות (!)

**דוגמה:**
```
קטגוריה      | Baseline | Reality | Gap
-------------|----------|---------|--------
קניות סופר  | 3,000 ₪  | 3,450 ₪ | +450 ₪ (15%)
מסעדות      | 1,500 ₪  | 2,200 ₪ | +700 ₪ (47%!) ⚠️
דלק          | 1,200 ₪  | 1,180 ₪ | -20 ₪  (OK)
```

**💡 זה חשוב!** המשתמש יגלה היכן הוא מבזבז בלי לשים לב.

## 2️⃣ **שלב 2: שינוי הרגלים (Behavior)**

### **🎯 מטרה:**
> התאמה אישית לדפוסי ההתנהלות של האדם  
> יצירת הרגלים חדשים שתואמים את המצב הפיננסי  
> חיזוק הרגלים יעילים והפחתת הרגלים שמובילים לבזבוז

**Phase:** `behavior`

---

### **מה המערכת עושה:**

#### **A. זיהוי דפוסים (Pattern Detection)**

**AI מחפש:**
- 🔄 **הוצאות חוזרות** - מנויים, חשבונות, הוראות קבע
- 📈 **מגמות** - עלייה/ירידה בקטגוריה לאורך זמן
- 📅 **דפוסים יומיים/שבועיים** - שישי = בילויים, ראשון = מכולת
- 🏪 **ספקים חוזרים** - אותו מקום = סיכוי גבוה לאותה קטגוריה
- 💳 **אמצעי תשלום** - כרטיס אשראי מסוים = קטגוריה מסוימת

**💾 שמירה:**
```sql
-- user_patterns
INSERT INTO user_patterns (user_id, pattern_type, pattern_key, pattern_value, confidence_score)
VALUES 
  ('user-123', 'merchant', 'רמי לוי', '{"category": "קניות סופר", "avg_amount": 250}', 0.95),
  ('user-123', 'day_of_week', 'friday', '{"category": "מסעדות", "avg_amount": 180}', 0.85),
  ('user-123', 'subscription', 'נטפליקס', '{"amount": 59, "frequency": "monthly"}', 1.0),
  ...
```

---

## 3️⃣ **שלב 3: תכנון תקציב (Budget)**

### **🎯 מטרה:**
> כל שורת הוצאה מקבלת תקציב מתוכנן  
> התאמה מלאה להכנסות ולצרכים האישיים  
> בניית מסגרת חודשית שניתן לעמוד בה לאורך זמן

**Phase:** `budget`

---

### **איך זה עובד:**

#### **שלב A: AI מחשב תקציב מומלץ**

**Input ל-AI:**
```typescript
{
  // מידע בסיסי
  totalIncome: 15000,
  fixedExpenses: 5000,    // שכירות, ביטוחים, וכו'
  availableBudget: 10000,
  
  // היסטוריה
  baseline: {             // מה המשתמש חשב
    'קניות סופר': 3000,
    'מסעדות': 1500,
    ...
  },
  
  reality: {              // מה קרה באמת
    'קניות סופר': 3450,
    'מסעדות': 2200,
    ...
  },
  
  // דפוסים
  patterns: [
    { pattern: 'overspending_restaurants', severity: 'high' },
    { pattern: 'consistent_groceries', severity: 'good' },
  ],
  
  // יעדים (אם כבר יש)
  goals: [
    { name: 'רכב חדש', target: 80000, deadline: '2027-11-01' }
  ]
}
```

```

  "categories": [
    {
      "name": "קניות סופר",
```

## 4️⃣ **שלב 4: תכנית יעדים ומטרות (Goals)**

### **🎯 מטרה:**
> הצבת מטרות ברורות עם תקציב ייעודי  
> הגדרת תאריך יעד  
> קביעת סדר עדיפויות  
> בחירת מקור התקציב הנכון לכל יעד

**Phase:** `goals`

---
