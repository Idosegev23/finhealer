## 🎯 ארבעת המרכיבים לבניית תקציב

### 1. ממוצע שורת הוצאה
- **מה זה?** ממוצע לפי שורת הוצאה ספציפית (לא קטגוריה כללית)
- **דוגמה:** "תקשורת" - לא כל "הוצאות קבועות"
- **מקור:** טבלת `transactions`
- **חישוב:** `AVG(amount) WHERE category = 'תקשורת'`

### 2. ממוצע חודשי
- **מה זה?** הממוצע מתוך 12 חודשים אחרונים
- **מטרה:** לזהות עונתיות ומגמות
- **מקור:** טבלת `transactions`
- **חישוב:** סכום 12 חודשים / 12

### 3. יעד (שנקבע יחד עם הלקוח)
- **מה זה?** הסכום שהלקוח רוצה להוציא
- **איך קובעים:** שיחה עם הלקוח, הבנת צרכים
- **מקור:** טבלת `budget_categories.target_amount`

### 4. המלצה (של גדי/AI)
- **מה זה?** המלצה מקצועית מבוססת ניתוח
- **כוללת:** טיפים לחיסכון, חבילות טובות יותר, אלטרנטיבות
- **מקור:** `behavior_insights` + AI

## 📐 פרמטרים לבניית תקציב נכון

### התקציב הנכון נבנה ביחס ל:

| פקטור | השפעה | איך מודדים |
|-------|--------|------------|
| **מספר נפשות** | גודל משפחה, הוצאות per capita | `user_financial_profile.children_ages` |
| **סוג מגורים** | שכירות/בעלות, גודל, מיקום | `user_financial_profile.fixed_rent` |
| **רמת הכנסה** | כמה נכנס, יציבות הכנסה | `user_financial_profile.salary` + `partner_salary` |
| **גובה התחייבויות** | הלוואות, משכנתא, ליסינג | `loans` + `pension_insurance` |
| **הוצאות משתנות** | התנהלות שוטפת, דפוסי צריכה | `transactions` WHERE `type='expense'` |
| **הוצאות קבועות** | שאינן התחייבויות (ביטוחים, מנויים) | `user_financial_profile.fixed_*` |
| **הוצאות מיוחדות** | אירועים, קניות גדולות | `expense_categories` WHERE `type='special'` |

## 🛡️ שני סוגי רשתות ביטחון

### 1. ביטחון פיננסי
| מרכיב | יעד | תיאור |
|-------|-----|--------|
| **קרן חירום** | 3-6 חודשי הוצאות | לאירועים בלתי צפויים |
| **חיסכון נזיל** | לפחות חודש אחד | נגיש מיידית |

### 2. ביטחון ביטוחי
| סוג ביטוח | חשיבות | הערות |
|-----------|---------|--------|
| **ביטוח חיים** | קריטי למפרנס | סכום = 10x הכנסה שנתית |
| **ביטוח בריאות** | חשוב מאוד | השלמות לקופ"ח |
| **אובדן כושר עבודה** | קריטי | 75% מההכנסה |
| **ביטוח דירה/רכב** | חובה | לפי חוק ושווי נכס |

## 🎯 סוגי יעדים

### יעד מכוון (ספציפי)
- **הגדרה:** סכום מוגדר + תאריך יעד
- **דוגמאות:**
  - רכב חדש: 80,000₪ בעוד שנתיים
  - חופשה משפחתית: 15,000₪ בעוד 6 חודשים
  - מקדמה לדירה: 200,000₪ בעוד 5 שנים
- **חישוב:** `target_amount / months_to_deadline = monthly_saving`

### יעד כללי (כיווני)
- **הגדרה:** שיפור מצב ללא סכום ספציפי
- **דוגמאות:**
  - שיפור מצב פיננסי
  - הקטנת חובות
  - הגדלת חיסכון
- **מדידה:** אחוזי שיפור, מגמות

```
   ↓

```

## 📊 מבנה DB לתקציב

### טבלת `budgets`
```sql
budgets {
  id: UUID,
  user_id: UUID,
  month: DATE,
  total_income: DECIMAL,
  total_fixed: DECIMAL,
  total_goals: DECIMAL,
  total_variable: DECIMAL,
  is_auto_generated: BOOLEAN,
  created_at: TIMESTAMP
}
```

### טבלת `budget_categories`
```sql
budget_categories {
  id: UUID,
  budget_id: UUID,
  category_id: UUID,
  avg_expense_line: DECIMAL,      -- ממוצע שורת הוצאה
  monthly_avg: DECIMAL,           -- ממוצע חודשי
  user_target: DECIMAL,           -- יעד לקוח
  gadi_recommendation: DECIMAL,   -- המלצת גדי
  allocated_amount: DECIMAL,      -- הקצאה סופית
  spent_amount: DECIMAL,          -- נוצל
  status: TEXT                    -- 'ok' / 'warning' / 'over'
}
```
