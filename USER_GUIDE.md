# φ Phi — מדריך שימוש

## איך מתחילים?

### שלב 1: הרשמה
- נכנסים ל-finhealer.vercel.app
- נרשמים עם Google או מייל+סיסמה
- מזינים מספר טלפון (וואטסאפ)

### שלב 2: שליחת דוח ראשון
φ ישלח הודעה בוואטסאפ. עכשיו צריך לשלוח דוח:

**מאיפה מורידים?**
- נכנסים לאפליקציה של הבנק
- דוחות → עו"ש → מורידים PDF (3 חודשים אחרונים מומלץ)
- שולחים את ה-PDF לצ'אט עם φ בוואטסאפ

**מה עוד אפשר לשלוח?**
- דוח פירוט כרטיס אשראי (ויזה, מסטרקארד, ישראכרט, כאל, max)
- תלוש משכורת (תמונה או PDF)
- כל מסמך פיננסי — PDF, תמונה, או אקסל

### שלב 3: אישור הסיווג
φ יסווג את כל התנועות אוטומטית ויציג סיכום:

```
✅ סיווגתי 47 תנועות!

✅ 42 — ודאות גבוהה:
🛒 סופר: 3,015₪ | 🏠 דיור: 4,200₪ | 🚗 רכב: 1,800₪

⚠️ 5 — כדאי לבדוק:
• מגדל 340₪ → ביטוח רכב
• העברה 2,000₪ → חיסכון
```

**אם הכל נכון** → לוחצים "הכל נכון ✅"
**אם יש טעות** → לוחצים "תיקון 🔧" → בוחרים קטגוריה → בוחרים תנועה → בוחרים חלופה. אפס הקלדה.

**φ לומד מכל תיקון** — בפעם הבאה יסווג נכון.

## מעקב הוצאות יומי

### רישום הוצאה בוואטסאפ
פשוט כותבים:
```
סופר 450
קפה 15
דלק 300
200 נעליים
```

φ מסווג, שומר, ומעדכן את התקציב. מחזיר אישור קצר:
```
✅ 450₪ סופר. השבוע 1,200₪ מזון מתוך 2,500₪ תקציב.
```

### ביטול הוצאה
```
בטל
```
מוחק את ההוצאה האחרונה.

## מה אפשר לבקש מ-φ?

| מה לכתוב | מה קורה |
|-----------|----------|
| **סיכום** | סיכום חודשי — הכנסות, הוצאות, יתרה |
| **גרף** | תרשים עוגה של הוצאות |
| **תקציב** | מצב תקציב — כמה נשאר, איפה חורגים |
| **יעדים** | התקדמות היעדים שלך |
| **תזרים** | תחזית 3 חודשים קדימה |
| **ציון** | ציון בריאות פיננסית (φ Score) |
| **ניתוח** | ניתוח דפוסי הוצאות — איפה אפשר לחסוך |
| **כפילויות** | בדיקת חשד לכפל תשלום |
| **עזרה** | תפריט מלא |

## הדשבורד (האתר)

כל מה שב-וואטסאפ — גם באתר, עם גרפים:

- **ראשי** — ציון φ, KPIs חודשיים, תקציב, יעדים, תנועות אחרונות
- **הוצאות** — פירוט לפי חודש וקטגוריה (קבועה/משתנה/מיוחדת)
- **הכנסות** — מקורות הכנסה + תנועות מהדוחות
- **תנועות** — רשימה מלאה עם חיפוש, סינון, עריכה
- **תקציב** — תכנון מול ביצוע + הפרש + ניצול %
- **יעדים** — כל היעדים + סימולציה
- **הלוואות** — פירוט + הצעת איחוד
- **סריקה** — העלאת מסמכים חדשים

**כפתור +** בתחתית הדשבורד → הוספת הוצאה מהירה

## טיפים

- **שלחו 3 חודשי דוחות** — ככל שיש יותר נתונים, φ חכם יותר
- **שלחו גם דוח בנק וגם דוח אשראי** — φ ימנע כפל ספירה אוטומטית
- **תקנו טעויות** — כל תיקון מלמד את φ. אחרי 2-3 תיקונים הוא כבר יודע
- **אל תדאגו מפורמט** — PDF, תמונה, אקסל — הכל עובד
- **רשמו הוצאות מזומן** — "קפה 15" — כי הן לא מופיעות בדוח הבנק
